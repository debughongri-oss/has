## Task 1: 改造 shared/auth.js 为 async + artist_profile 权威源
**Requirement:** SEC-04
**Files:**
- `cloudfunctions/shared/auth.js`（根模板）
- `cloudfunctions/profile/shared/auth.js`
- `cloudfunctions/bookings/shared/auth.js`
- `cloudfunctions/services/shared/auth.js`
- `cloudfunctions/works/shared/auth.js`

**Changes:**
1. 移除 `const ARTIST_OPENID = '...'` 硬编码
2. `requireArtist` 改为 `async function requireArtist(wxContext, db)`，签名增加 `db` 参数（调用方传入已初始化的 db 实例，避免每个 shared/auth.js 各自 cloud.init）
3. 函数体：
   ```javascript
   async function requireArtist(wxContext, db) {
     try {
       const { data } = await db.collection('artist_profile').limit(1).get()
       if (data.length === 0) return { ok: false, response: { errCode: -1, errMsg: '化妆师资料未初始化' } }
       if (wxContext.OPENID === data[0]._openid) return { ok: true }
       return { ok: false, response: { errCode: -1, errMsg: '无权限操作' } }
     } catch (err) {
       console.error('requireArtist 查询失败:', err)
       return { ok: false, response: { errCode: -1, errMsg: '身份验证失败' } }
     }
   }
   ```
4. `sanitizeProfileUpdate` 保持不变
5. `module.exports = { requireArtist, sanitizeProfileUpdate }`（签名不变，仅函数变为 async）
6. 5 份文件内容完全一致（复制粘贴）

**Verification:**
- `node -e "const m=require('./cloudfunctions/shared/auth.js'); console.log(typeof m.requireArtist)"` 应输出 `function`
- grep 确认 5 份 shared/auth.js 中无 `ARTIST_OPENID` 残留：`grep -r ARTIST_OPENID cloudfunctions/*/shared/` 应无匹配
