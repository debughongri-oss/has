**3. miniprogram/pages/admin/profile/edit.js:**
- 在文件顶部 require 中添加 `const { STYLE_TAGS } = require('../../../utils/constants')`
- data 中添加：
  - `experienceYears: ''` — 从业年数（字符串，保存时转数字）
  - `selectedStyleTags: []` — 已选风格标签数组
  - `styleTagOptions: []` — 所有可选标签（onLoad 时从 STYLE_TAGS 转换）
- onLoad 或 loadProfile 中添加：
  - `styleTagOptions: STYLE_TAGS.map(t => ({ ...t, selected: false }))`
- loadProfile 的 setData 中添加：
  - `experienceYears: data.experience_years ? String(data.experience_years) : ''`
  - `selectedStyleTags: data.style_tags || []`
  - 同时更新 styleTagOptions 的 selected 状态：`styleTagOptions: STYLE_TAGS.map(t => ({ ...t, selected: (data.style_tags || []).includes(t.label) }))`
- 添加新方法：
  - `onExperienceYearsInput(e)`: `this.setData({ experienceYears: e.detail.value })`
  - `onToggleStyleTag(e)`: 获取 `e.currentTarget.dataset.label`，在 selectedStyleTags 中切换（有则移除，无则添加），同时更新 styleTagOptions 的 selected 状态
- saveProfile 中：
  - 在 saveData 对象中添加 `experience_years: parseInt(experienceYears) || 0`
  - 添加 `style_tags: this.data.selectedStyleTags`
  - Per D-02: 保留原有 `experience` 文本字段
