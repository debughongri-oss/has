# Requirements: 化妆师个人作品展示与预约小程序

**Defined:** 2026-04-17
**Core Value:** 客户看到作品后能直接预约化妆服务——从"看到好看的作品"到"我要预约"的路径最短

## v1 Requirements

### Authentication

- [x] **AUTH-01**: User can silently log in via WeChat (wx.login) and obtain unique identity (OpenID) — ✓ v1.0
- [x] **AUTH-02**: User can view makeup artist profile (name, avatar, bio, specialties, experience) — ✓ v1.0

### Portfolio

- [x] **PORT-01**: Artist can create portfolio items with multiple images and description — ✓ v1.0
- [x] **PORT-02**: Artist can edit own portfolio items — ✓ v1.0
- [x] **PORT-03**: Artist can delete own portfolio items — ✓ v1.0
- [x] **PORT-04**: User can browse portfolio items in a paginated image grid with lazy loading — ✓ v1.0
- [x] **PORT-05**: User can view portfolio item detail with multi-image carousel (long-press to save) — ✓ v1.0
- [x] **PORT-06**: User can filter portfolio items by category (bridal, daily, engagement, custom tags) — ✓ v1.0
- [ ] **PORT-07**: User can view before/after comparison with interactive slider on portfolio detail — ⚠️ Deferred

### Services

- [x] **SERV-01**: Artist can create service items with name, duration estimate, price range, and description — ✓ v1.0
- [x] **SERV-02**: Artist can edit service items — ✓ v1.0
- [x] **SERV-03**: Artist can delete service items — ✓ v1.0
- [x] **SERV-04**: User can browse all service items in a card-style list — ✓ v1.0

### Booking

- [x] **BOOK-01**: User can select a service and submit a booking request with date and time — ✓ v1.0
- [x] **BOOK-02**: User can view artist's available time slots on a calendar before booking — ✓ v1.0
- [x] **BOOK-03**: User can attach a note/message when submitting a booking — ✓ v1.0
- [x] **BOOK-04**: Artist can review booking requests and accept, reject, or propose reschedule — ✓ v1.0
- [x] **BOOK-05**: Artist can add internal notes to bookings (visible only to artist) — ✓ v1.0
- [ ] **BOOK-06**: User receives subscription message notification when booking status changes — ⚠️ Requires manual WeChat template config
- [ ] **BOOK-07**: Artist receives notification when new booking is submitted — ⚠️ Requires manual WeChat template config
- [x] **BOOK-08**: User can view own booking history with status badges — ✓ v1.0

### Management

- [x] **MGMT-01**: Artist can access admin panel within mini program to manage portfolio, services, and bookings — ✓ v1.0
- [x] **MGMT-02**: User can share portfolio items to WeChat chat and Moments — ✓ v1.0
- [ ] **MGMT-03**: Artist can generate a mini program QR code poster for marketing — ⚠️ Deferred
